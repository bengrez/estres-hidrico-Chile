# Pedagogical approach

- **Audience:** Students from 7º básico to IVº medio.
- **Bilingual:** Every string is available in Spanish and English via `react-i18next`.
- **Levels:** The Explore view includes a selector for Basic/Intermediate/Advanced text density so teachers can adapt explanations.
- **Tooltips and summaries:** Each indicator includes a short description, and the backend provides quick summary texts in both languages.
- **Activities:** The Learn view contains cards with core concepts, a glossary, and a short quiz for classroom discussions.
